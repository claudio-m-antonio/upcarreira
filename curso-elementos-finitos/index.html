Objetivo do Curso

Com este curso você aprenderá a fazer simulação estrutural possibilitando o estudo do comportamento de componentes ou montagens submetidos às condições e carregamentos de uso para análise e otimização.

Este processo fornece agilidade no desenvolvimento e modificações de projetos, pois tudo é testado antes do projeto final, garantindo a qualidade, eficiência e segurança do produto final.

A aplicação desta tecnologia, cada vez mais demandada em diversas áreas como engenharia civil, estrutural, industrial, medicina, odontologia entre outras, requer profissionais que estejam capacitados na utilização dos softwares como o Abaqus através do método numérico dos Elementos Finitos.

Neste curso você aprenderá o método dos Elementos Finitos e a utilização do Software Abaqus (será fornecido no curso o link para download da versão estudante).
